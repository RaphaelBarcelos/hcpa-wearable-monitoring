# Projeto Acurácia de Wearables
Pesquisa conduzida no Hospital de Clínicas com o objetivo de desenvolver um modelo de inteligência artificial capaz de realizar predições clínicas a partir de dados coletados por dispositivos wearables

# Sobre o projeto
O proojeto investiga a acurácia de wearables como fonte de dados para modelos preditivos em saúde. As predições abrangem desde o monitoramento de atividade física até a detecção de eventos clínicos como arritmias, com foco em validar a viabilidade do uso desses dispotisivos em contextos hospitalares e de acompanhamento remoto de pacientes.

# Datasets
Os modelos são treinados com datasets públicos
Datasets: 
PAMAP2 Physical Activity Monitoring dataset

# Tecnologias Utilizadas
